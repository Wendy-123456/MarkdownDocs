## 1. 创建和管理文件

### 新建文件
- **通过导航栏**：点击左侧导航栏上的新建文件图标。
- **通过界面中间**：点击界面中间的“创建新文件”。
- **通过快捷键**：使用 `Ctrl+N`（Windows/Linux）或 `Cmd+N`（Mac）快捷键。

### 打开文件
- **通过导航栏**：点击左侧文件列表中的文件名。
- **通过搜索**：点击左侧边栏上方的放大镜图标，输入文件名或关键词，回车查看搜索结果。

## 2. 编辑和格式化文本

### 基本 Markdown 语法

#### 标题
使用 `#` 创建不同级别的标题，例如：
- `# 一级标题`
- `## 二级标题`
- `### 三级标题`

#### 强调文本
- **加粗**：使用 `**` 或 `__`，例如 `**加粗文本**`。
- **斜体**：使用 `*` 或 `_`，例如 `*斜体文本*`。
- **删除线**：使用 `~~`，例如 `~~删除线文本~~`。
- **高亮**：使用 `==`，例如 `==高亮文本==`。

#### 列表
- **无序列表**：使用 `-`、`+` 或 `*`，例如 `- 项目A`。
- **有序列表**：使用数字和 `.`，例如 `1. 第一步`。

#### 链接
- **内部链接**：使用 `[[`，例如 `[[文件名]]`。
- **外部链接**：使用 `[链接文字](链接地址)`，例如 `[百度](https://www.baidu.com)`。

### 特殊格式
#### 代码块
使用三个反引号 `````，例如：
```python
print("Hello, World!")
```
#### 表格
使用 `|` 和 `-`，例如：
```text
| 表头1 | 表头2 |
|-------|-------|
| 内容1 | 内容2 |
| 内容3 | 内容4 |
```

## 3. 查看生成格式和原始文档

### 预览模式
- **切换预览模式**：点击标题栏的预览按钮，或使用快捷键 `Ctrl+E`（Windows/Linux）或 `Cmd+E`（Mac）。

### 查看原始文档
- **切换编辑模式**：点击标题栏的编辑按钮，或使用快捷键 `Ctrl+E`（Windows/Linux）或 `Cmd+E`（Mac）。

## 4. 常用快捷键

- **新建文件**：`Ctrl+N`（Windows/Linux）或 `Cmd+N`（Mac）
- **保存文件**：`Ctrl+S`（Windows/Linux）或 `Cmd+S`（Mac）
- **预览/编辑切换**：`Ctrl+E`（Windows/Linux）或 `Cmd+E`（Mac）
- **搜索文件**：`Ctrl+P`（Windows/Linux）或 `Cmd+P`（Mac）
- **查找文本**：`Ctrl+F`（Windows/Linux）或 `Cmd+F`（Mac）

## 5. 插件推荐

### Citation
- **功能**：帮助学术写作者管理引用和生成文献引用表。
- **安装**：在 Obsidian 的插件市场中搜索并安装 `Citation` 插件。

### Pandoc
- **功能**：将 Markdown 文档转换为其他格式，如 PDF、Word 等。
- **安装**：在 Obsidian 的插件市场中搜索并安装 `Pandoc` 插件。

### Self-Hosted LiveSync
- **功能**：实现 Markdown 文档的实时同步，支持与 CouchDB 数据库的连接。
- **安装**：在 Obsidian 的插件市场中搜索并安装 `Self-Hosted LiveSync` 插件。

## 6. 常见问题及解决方法

### 无法打开文件
- **检查文件路径**：确保文件路径正确，文件存在于指定目录中。
- **检查文件格式**：确保文件格式为 `.md` 或 `.markdown`。

### 格式显示不正确
- **检查 Markdown 语法**：确保使用正确的 Markdown 语法。
- **重启 Obsidian**：关闭并重新打开 Obsidian，有时可以解决格式显示问题。

### 插件不工作
- **检查插件状态**：确保插件已启用，未被禁用。
- **更新插件**：检查是否有插件更新，更新到最新版本。
- **查看日志**：在 Obsidian 的设置中查看插件日志，查找错误信息。
